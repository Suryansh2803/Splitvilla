# Splitvilla
💸 Splitvilla – Smart Expense Splitter

Splitvilla is a simple yet powerful expense splitting web application designed to minimize the number of transactions between users.

Instead of multiple confusing payments, Splitvilla calculates the most efficient way to settle debts with the least number of transactions.

🌟 Core Feature: Minimal Transactions

💡 Splitvilla doesn’t just split expenses — it optimizes settlements

Reduces unnecessary payments between users
Calculates the minimum number of transactions required
Ensures users pay as little as possible, as efficiently as possible
Eliminates circular and redundant payments
🧠 Example

Instead of:

A → B ₹500
B → C ₹500

👉 Splitvilla simplifies it to:

A → C ₹500

✔️ Fewer transactions
✔️ Less confusion
✔️ Faster settlement
